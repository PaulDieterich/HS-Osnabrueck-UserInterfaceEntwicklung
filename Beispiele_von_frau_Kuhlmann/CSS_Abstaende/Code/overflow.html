Fender Stratocaster

Die Fender Stratocaster oder "Strat" ​​ist eine der beliebtesten E-Gitarren aller Zeiten. Ihr Design wurde von vielen Gitarrenherstellern nachgebildet. Es wurde 1954 von Leo Fender, George Fullerton und Fredie Tavares entworfen..

Gibson Les Paul

Die Gibson Les Paul ist eine Solid-Body-E-Gitarre, die erstmals 1952 verkauft wurde. Die Les Paul wurde von Ted McCarty in Zusammenarbeit mit dem bekannten Gitarristen Les Paul entworfen, den Gibson für das neue Modell unterstützte. Es ist eine der bekanntesten E-Gitarren-Typen der Welt.
